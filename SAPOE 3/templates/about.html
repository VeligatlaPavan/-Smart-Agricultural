About OptiCrop

Smart Agricultural Production Optimization Engine (OptiCrop) is a data-driven agronomy advisor powered by Machine Learning.

Project Objective

Farmers face significant uncertainty due to changing environmental conditions, depleting soil nutrition, and shifting weather patterns. OptiCrop aims to mitigate this risk by applying a lightweight, fast, and highly accurate classification model that matches the chemical parameters of the soil (N, P, K, pH) and the ambient environment (Temperature, Humidity, Rainfall) to the crop profile that will yield the highest probability of success.

This is designed as a college demonstration project, showcasing the practical integration of Python-based Machine Learning models (Scikit-Learn) with standard web frameworks (Flask) to produce a fast, responsive, and easy-to-understand farm helper.

Features & Input Boundaries

The system evaluates 7 distinct inputs to calculate suitability matches. The bounds are set based on agricultural feasibility:

Feature Unit Feasible Range Primary Agricultural Role
Nitrogen (N) mg/kg (ppm) 0 - 140 Leaf growth, chlorophyll formation
Phosphorus (P) mg/kg (ppm) 5 - 145 Root structure, flowering, early cell division
Potassium (K) mg/kg (ppm) 5 - 205 Disease resistance, water movement control
Temperature °C 10.0 - 50.0 Determines growth speed & metabolic activity
Humidity % 15.0 - 100.0 Controls transpiration rate (water loss)
pH Level pH Scale 3.5 - 10.0 Dictates nutrient uptake capability in roots
Rainfall mm 20.0 - 300.0 Determines irrigation requirement

Machine Learning Core

We implement an ensemble decision classifier using the Random Forest Algorithm. Unlike a single Decision Tree which is prone to overfitting, Random Forest aggregates predictions from multiple decision paths to ensure stability.

  • Algorithm Random Forest
  • Estimators (Trees) 50
  • Classes (Crops) 22
  • Max Tree Depth 12
  • Features Used 7
  • Target Accuracy > 99%
  • Model File Size ~250 KB
